# 🍔 pnpm

### 🍔 pnpm 是什么？

pnpm 和 npm/yarn 一样是用来管理包工具的。但是它比后两者优秀。👇  

- [pnpm 是凭什么对 npm 和 yarn 降维打击的](https://zhuanlan.zhihu.com/p/549189023)


## 🔨 安装

```sh 
npm i -g pnpm
```

## 🔧 常用命令

它的命令和npm一致，对比 yarn 的反人类（比如npm安装是用 `install` ，yarn用 `add` )

1. pnpm init 
2. pnpm install 
3. pnpm uninstall 
4. pnpm run 
5. .....
